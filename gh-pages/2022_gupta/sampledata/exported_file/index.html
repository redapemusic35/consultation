Doccano Deployment Tutorial for Meha Gupta Sentiment Annotation Project

  1. This requires git, github, and docker command-line interface

  2. Navigate to github.com/doccano/doccano

  3. fork github.com/doccano/doccano

  4. git clone your fork of doccano

  5. cd into cloned project

  6. create .env in root of cloned project with the following contents:

    # platform settings
ADMIN_USERNAME=admin
ADMIN_PASSWORD=password
ADMIN_EMAIL=admin@example.com

# rabbit mq settings
RABBITMQ_DEFAULT_USER=doccano
RABBITMQ_DEFAULT_PASS=doccano

# database settings
POSTGRES_USER=doccano
POSTGRES_PASSWORD=doccano
POSTGRES_DB=doccano

  1. run this command in your terminal docker-compose -f docker/docker-compose.prod.yml --env-file .env up

  2. then access [127.0.0.1/](http://127.0.0.1/)

  3. Inspect to see working

  4. then click "Deploy to Heroku" button.

  5. will take you to heroku, create free account

  6. go back to heroku and click deploy button again

  7. site will run after 10 minutes or so
